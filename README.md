# Kickboard_violation_detection

**시기**

- 2024.10~2024.11

**본인 역할 및 기여**

- AI-hub, kaggle 등에서 데이터를 확보하고 그에 맞는 프로젝트 기획 (PM)
- 모델 성능 향상을 위한 이미지 데이터를 전처리, 라벨링 작업
- 상품화 기획

**활용 데이터**

- 킥보드 이미지 데이터, 헬멧 이미지 데이터
    - kaggle, AI-hub, 웹 이미지 검색 등에서 다양하게 수집

**사용 기술 스택**

- Azure, CustomVision, Plask

**성과**

- 프로젝트 기획 및 리더십 강화
- 데이터 수집 및 전처리 역량 향상
